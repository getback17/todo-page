# 📋 ToDo App на React

Мини-проект для тренировки React-хуков и работы с localStorage. Простой список задач с возможностью добавлять, удалять и отмечать выполненные. Есть доп. функции: переключение между светлой и тёмной темой, а также смена языка (русский/английский), чтобы было интереснее. Все задачи сохраняются в браузере, так что после перезагрузки страницы они не пропадают.

## Стек
- **React**
- **CSS3**
- **localStorage**

## Как запустить локально
1. Клонируй репозиторий 
2. Перейди в папку проекта 
3. Установи зависимости:  
   `npm install`
4. Запусти приложение:  
   `npm start`

После этого откроется страница с ToDo-списком.
