# ToDo App на React

Мини-проект для демонстрации React-хуков и работы с localStorage. Простой список задач с возможностью добавлять, удалять и отмечать выполненные. Есть доп. функции: переключение между светлой и тёмной темой, а также смена языка (русский/английский), чтобы было интереснее. Все задачи сохраняются в браузере, так что после перезагрузки страницы они не пропадают.

## Стек
- **React**
- **CSS3**
- **localStorage**

## Как запустить локально
1. Клонируйте репозиторий:
    `git clone https://github.com/getback17/todo-page`
2. Перейдите в папку проекта:
    `cd todo-page`
3. Установите зависимости:  
   `npm install`
4. Запустите приложение:  
   `npm start`

После этого откроется страница с ToDo-списком.
